## Rutas  

###GET

####Lista programaciones 
Listar programaciones existentes
http://localhost:8080/ProgramacionApi/programacion		

####Listar Peliculas 
Lista las peliculas disponibles para la programacion 
http://localhost:8080/ProgramacionApi/pelicula			//todas la peliculas

####Busqueda Pelicula
Busca peliculas por id
http://localhost:8080/ProgramacionApi/pelicula/pe1		//una pelicula (pe1 a pe5)

####Listar Series 
Lista las series disponibles  
http://localhost:8080/ProgramacionApi/serie				//todas la series

####Busqueda Series
Busca series por id
http://localhost:8080/ProgramacionApi/serie/se1			//una serie (se1 a se5)

###Lista Capitulos
Dada una serie lista todos los capitulo
http://localhost:8080/ProgramacionApi/serie/se1/capitulo	//capitulos de una serie

###Busqueda Capitulo dada una serie
Dada una serie busca un capitulo de esa serie
http://localhost:8080/ProgramacionApi/serie/se2/capitulo/se2T1E2	//un capitulo (serie 2 capitulo: Temporada 1,Episodio 1)
